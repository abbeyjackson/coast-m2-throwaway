# Greeter — How This Works

Greeter is a tiny helper that says hello.

- If you give it your name, it replies with "Hello, `<your name>`!"
- If you don't give it a name, it replies with "Hello, world!"
- If you give it an empty name, it still replies politely: "Hello, !"

That's it — no complicated logic, just a friendly greeting building block that other parts of the app can reuse.
