# Greeter

## What it does

`Greeter` says hello. Give it a name and it greets that person; don't give it a name and it uses a friendly default.

## How it works

- `greeter.greet(name: "Ada")` returns `"Hello, Ada!"`
- `greeter.greet()` returns `"Hello, world!"`

That's it — no configuration, no external services, just a small piece of reusable code other parts of the app can call whenever they need to greet someone.
